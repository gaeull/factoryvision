# 쓰러진 사람을 감지하고 모니터링할 수 있는 웹 솔루션


## 기능 소개

  - 영상에서 분석하여 넘어짐이 인식되면 그에 대한 정보를 알림
  - 게시판 기능으로 위험 발생을 알림
  - 비상 호출 버튼으로 위험 상황을 관리자에게 알림
